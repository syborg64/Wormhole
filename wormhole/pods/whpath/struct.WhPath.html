pub fn join<T>(&self, segment: &T) -> Self
where T: JoinPath + ?Sized,

Join the current path with a new segment. If the segment starts with a / or ./ or ../, the leading slash is removed. If the current path is empty, the segment is added as is. If the current path is not empty, the segment is added after adding a slash at the end of the current path. If the segment is hidden (is_hidden() return true), the segment is added as is, without adding a slash.
